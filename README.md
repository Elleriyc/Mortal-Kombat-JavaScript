# Mortal-Kombat-JavaScript
Version 1.0

The famous Mortal Kombat3 Ultimate game implemented in JavaScript for your pleasure :) 
There are still improvements to be made, which will of course be added over time :) 
Good game ;) 

Play Controls : 

Sub Zero :                                 
a : backup  
d : forward 
r : punch                    
e : kick
t : iceball
f : friendship 

Kano : 
<- : forward 
-> : backup
i : punch                       
o : kick
p : fireball
u : babality
